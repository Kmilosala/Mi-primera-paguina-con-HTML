#  Primera Página Web – Fórmula 1 (HTML)

Este repositorio contiene mi **primera página web** desarrollada en **HTML y CSS**, creada como parte de un proyecto académico durante mis primeros semestres de **Ingeniería de Sistemas**. El objetivo principal fue aplicar los fundamentos iniciales del desarrollo web y publicar por primera vez un proyecto en **GitHub**.

---

##  Descripción del Proyecto

La página está dedicada a la **Fórmula 1**, uno de los deportes de motor más importantes del mundo. A través de contenido informativo, imágenes y secciones temáticas, el proyecto presenta:

* Historia general de la Fórmula 1
* Pilotos históricos destacados
* Escuderías actuales
* Ranking personal de pilotos favoritos

Este proyecto representa el **primer acercamiento práctico** al desarrollo web y a la estructuración de contenido en HTML.

---

##  Objetivo Académico

* Aplicar conceptos básicos de **HTML5**
* Introducir estilos simples con **CSS**
* Comprender la estructura de una página web
* Publicar un proyecto en un repositorio de GitHub
* Desarrollar habilidades iniciales de documentación y presentación

---

##  Tecnologías Utilizadas

* HTML5
* CSS3 (estilos embebidos)
* Editor de código: Visual Studio Code

---

##  Contenido de la Página

*  Página informativa sobre la Fórmula 1
*  Sección histórica de pilotos legendarios (Fangio, Lauda, Senna, Schumacher)
*  Información sobre escuderías actuales (Mercedes, Red Bull, Ferrari, McLaren, entre otras)
*  Top personal de pilotos favoritos
*  Uso de imágenes externas para reforzar el contenido visual

---

## Estructura del Proyecto

El proyecto está compuesto por un único archivo HTML que contiene:

* Estructura básica del documento (`<!DOCTYPE html>`)
* Uso de etiquetas semánticas y estructurales
* Estilos aplicados directamente en el archivo
* Organización del contenido por secciones

Este enfoque corresponde a una **etapa inicial de aprendizaje**, previa al uso de frameworks o separación de capas.

---

##  Aprendizajes Clave

* Comprensión de la estructura básica de una página web
* Uso correcto de etiquetas HTML
* Inclusión de imágenes y contenido multimedia
* Primer contacto con estilos CSS
* Introducción al uso de GitHub como portafolio
